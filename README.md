# Focus Planner – Task Manager for ADHD

This repository hosts the Privacy Policy for the Focus Planner mobile application.

## 📱 About the App
Focus Planner is an ADHD-friendly task management app designed to help users stay organized, improve focus, and boost productivity.

Key features include:
- Daily task planning
- Focus timer (Pomodoro)
- Reminders and notifications
- Progress tracking with streaks
- Simple and distraction-free design
